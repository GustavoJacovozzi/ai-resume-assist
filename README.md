# AI Resume Assist

Sistema inteligente para análise de currículos utilizando IA Generativa.

## Tecnologias

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Google Gemini
- PyMuPDF

## Funcionalidades

- Cadastro de currículos
- Upload de PDF
- Extração automática do texto
- Análise utilizando IA
- Comparação entre currículo e vaga
- API documentada via Swagger

Projeto em desenvolvimento.
